Classroom Management System

This project aims to develop an attractive multi-user meeting scheduler web app for tracking meetings and appointments in an organization's office. Users will have separate user IDs and passwords for login and logout. They can add meeting details such as title, description, date, and time, and schedule meetings for specific time slots and meeting rooms. The application will notify users if a slot is unavailable and suggest alternative rooms. If no slots are available, it will display blocking meetings and responsible users. Users can invite attendees and receive notifications. An admin view will show all meetings in all rooms.

Code : MeetingPlanner
